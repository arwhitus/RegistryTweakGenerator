Registry Tweak Generator
  • About/GitHub

Generate .reg files to apply tweaks to the Windows 10 registry.

All keys modified in the registry can be viewed by clicking 'show change' on the respective tweak. The produced .reg file can be viewed with any text editor to view the changes, and will be commented for ease of use. This software is provided with no warranty expressed or implied. Issues and recommendations can be submitted via GitHub by following the link in the top bar.

  • 1. Select the tweaks you wish to apply (or reverse).
  • 2. Click the 'generate' button at the top of the page.
  • 3. Double click on the downloaded file to run it.
  • 4. Some tweaks may require a system restart to apply.
